# Usage
<!-- usage -->
```sh-session
$ npm install -g wc-cli
$ wc-base COMMAND
running command...
$ wc-base (-v|--version|version)
wc-cli/0.0.1 win32-x64 node-v11.12.0
$ wc-base --help [COMMAND]
USAGE
  $ wc-base COMMAND
...
```
